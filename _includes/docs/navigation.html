The Navigation

The file _includes/nav.html contains the code for your site's navigation. The code is quite simple.

Each navigation link is included inside an <li> tag. For instance, the code for the Archives page, is the following: 

<li>
  <a href="/archives"> <!-- Place the link inside the double quotes -->
    Archives <!-- The text of the link -->
    <div class="border1"></div>
    <div class="border2"></div>
  </a>
</li>

Useful comments can be found in the actual file as well
