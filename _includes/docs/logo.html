The Logo

You can customize the site's logo by opening _includes/logo.html

Once there, you can change the text of the logo, use an image, or add whatever you want.

If you have an image, place the file preferably in the images folder and just use an <img> tag. For example, if your image file is named logo.png, your entire logo.html file will look like this: 

<section class="site-logo">
   <a class="site-title-link" href="/">
       <img src="/images/logo.png" alt="Logo alt text">
       <div class="a-left-line"><div class="runner"></div></div>
   </a>
   <div class="site-slogan">A Jekyll Theme</div>
   <div class="a-right-line"></div>
</section>

YB also offers you the option to have a site slogan. You can change it by modifying the text in the div with the class name of site-slogan. If you do not want to use a slogan, just delete the following lines altogether: 

<div class="site-slogan">A Jekyll Theme</div>
<div class="a-right-line"></div>
